Add as many screenshots as you like — each one's players are added to the list below. Reading happens on the server with offline OCR, so the first one can take ~20–30s while it warms up.

Saved rankings

Your ranked lists are kept here between visits — they won't disappear when you reload. Load one to edit it or convert it to Underdog; Update replaces it in place, or Save as new keeps a separate copy.

2. Check & fix

Rank follows the order players are listed — the first is #1. Position rank (QB5, RB12…) is figured automatically. Edit any cell so names match across sites.

Team and position are filled in automatically from each player's name (current NFL rosters), so you usually won't need this. Use it for anyone left blank — e.g. a rookie or an OCR misread: pick a position and Apply to blank rows sets only the empty ones.

    Send to Yahoo

    Yahoo has no rankings upload, so this builds a one-click bookmarklet from your list above. Drag it to your bookmarks bar once, then open Yahoo's Edit Pre-Draft Player Rankings page and click it — it matches your players to Yahoo's list and moves them into Preferred Players in your order. You review and press Yahoo's Save Changes.

    ★ All22 → Yahoo

    1. Drag the ★ All22 → Yahoo button above onto your browser's bookmarks bar.
    2. On Yahoo Fantasy: Draft → Pre-Draft Rankings → Edit My Pre-Draft Player Rankings.
    3. Click the bookmark. A small All22 panel appears — press Set my rankings.
    4. Check the order in Preferred Players, then press Yahoo's Save Changes.

    If players don't move automatically (Yahoo changes this page often), use the panel's Teach the ▶ button, or Copy diagnostics and send them back here so this can be tuned. The panel always lets you copy your list in order as a fallback.
